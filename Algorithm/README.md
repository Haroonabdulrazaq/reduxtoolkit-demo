# Big O

## Object Big O

- Insert - O(1)
- Remove - O(1)
- Access - O(1)
- Search - - O(n)

### Properties on Object

- Obj.keys() - O(n)
- Object.values() - O(n)
- Object.entries() - O(n)

## Array

- Insert/ remove at end - O(1)
- Insert/ remove at begining - O(n)
- Access - O(1)
  Searching - O(n)

### Properties on Array

- push()/ pop() - O(1)
- shift()/unshift() /splice() /slice() /concat() - O(n)
- forEach()/ map()/ reduce()/ filter() - O(n)
